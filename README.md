# PySlice3D

A lightweight Python library for 3D volume visualization.

## Quickstart

```sh
git clone https://github.com/GeorgeWilliamStrong/PySlice3D
pip install -e .
```

## Usage
```python
from pyslice import slicer
slicer(...)
```
